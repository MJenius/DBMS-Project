Query Parameters

This nested query finds customers who have more than X active orders in the system.

View SQL Query 

SELECT Customer_ID, CONCAT(First_Name, ' ', Last_Name) AS CustomerName, Email, Phone_No
FROM customers
WHERE Customer_ID IN (
    SELECT Customer_ID FROM customer_current_orders
    GROUP BY Customer_ID
    HAVING COUNT(*) > {{ min_orders }}
)
ORDER BY Customer_ID
